# Prodigy InfoTech Generative AI Internship - Task 4

## Task Name
Image-to-Image Translation using Pix2Pix

## Objective
Implement an image-to-image translation model using Pix2Pix (Generative Adversarial Network).

## Description
In this project, Pix2Pix (a GAN-based model) is used for image-to-image translation. 
The model learns to convert input images into corresponding output images using paired datasets.

Pix2Pix uses:
- Generator
- Discriminator
- Adversarial Training

## Tools & Technologies Used
- Python
- TensorFlow / PyTorch
- GAN (Pix2Pix)
- Google Colab

## How It Works
1. Load paired dataset.
2. Train generator and discriminator models.
3. Generate translated output images.
4. Compare input and generated output.
